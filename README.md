ARPG Draft Builder

A procedural Action RPG loot draft prototype built in Python.

Players draft gear, build a character loadout, and attempt to survive
25 arena rounds of escalating enemies.

Features:
- Procedural item generation
- Multiple gear slots
- Rarity and tier system
- Boss encounters
- Draft-based loot selection
- GUI built with Tkinter
- Persistent best-run tracking

Controls:
- Choose a class
- Draft gear
- Survive combat rounds
- Replace items or skip rewards

Future ideas:
- seeded runs
- item art
- deeper combat
- ability systems
